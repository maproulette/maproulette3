# Loading States

## LoadingOverlay (`src/components/shared/LoadingOverlay.tsx`)

A semi-transparent overlay with a centered spinner and message. Covers its parent container during async operations.

## GridSkeleton (`src/components/shared/GridSkeleton.tsx`)

A grid of animated skeleton cards that match the layout of actual content, shown while data is loading.

## Loader (`src/components/ui/Loader.tsx`)

An animated logomark with an optional text message. Can be displayed inline or centered in the full viewport.
