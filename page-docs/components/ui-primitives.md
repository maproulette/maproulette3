# UI Primitives

**Source**: `src/components/ui/`

Low-level UI components built on Radix UI and styled with Tailwind CSS. These are the building blocks used across all pages.

- **Button** — Multiple variants (default, destructive, outline, secondary, ghost, link) and sizes
- **Card** — Content container with header, content, and footer sections
- **Badge** — Pill-shaped label for tags, statuses, and counts
- **Dialog** — Modal overlay with header, content, and footer
- **Drawer** — Slide-up panel, closeable with ESC
- **ScrollArea** — Custom-styled scrollable container
- **Input** — Text input with file and error state support
- **Resizable** — Draggable split-panel layout (ResizablePanelGroup, ResizablePanel, ResizableHandle)
- **Tabs** — Tabbed content with trigger list and panels
- **Tooltip** — Hover tooltip with portal rendering
- **Alert** — Status message (default, destructive, warning)
- **Empty** — Empty state display with icon, title, and description
- **Select, Checkbox, RadioGroup, Switch, Toggle, ToggleGroup** — Form controls
- **Textarea, Label, Field, Form, InputGroup, ButtonGroup** — Form layout
- **Table, Pagination, Separator** — Data display
- **Popover, Collapsible** — Expandable content
- **AlertDialog** — Confirmation modal
- **Skeleton, Spinner, Loader, Logomark, BackLink, Progress** — Utility components
