# ChallengeCard

**Source**: `src/components/shared/ChallengeCard.tsx`

A clickable card that displays a challenge's name, project, tasks remaining, completion progress bar, difficulty badge, and last updated date. A colored sidebar on the right edge indicates completion status (red < 25%, yellow 25-50%, orange 50-90%, blue 90%+). Links to the challenge's browse page.

**Used in**: Explore Challenges, Browse Project, Manage Project Detail.
