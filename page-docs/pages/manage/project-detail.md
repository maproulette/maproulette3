# Manage Project Detail (`/manage/project/$projectId`)

**Source**: `src/components/ManagementPages/ManageProjectDetail/`

Detail view for a single managed project. Shows the project name, status, and description in the header with edit, create challenge, and action dropdown buttons.

Displays stats cards for total challenges, enabled challenges, and total tasks. Below is a searchable grid of the project's challenges with management actions on each card.
