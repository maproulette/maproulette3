# Manage Challenges (`/manage/challenges`)

**Source**: `src/components/ManagementPages/ManageChallenges/`

A searchable grid of all the user's challenges across all projects. Each card shows the challenge name, ID, description, status badge, tasks remaining, completion progress bar, and difficulty level. Clicking a card navigates to its detail page.

A "Create New Challenge" button links to the challenge creation form.
