# Settings (`/settings`)

**Source**: `src/components/SettingsPage/`

User account settings organized into four tabs. Requires authentication.

### General

Language, default editor, default basemap, and custom basemap URL.

### Notifications

Email address and notification subscription preferences.

### Plugins

Two sub-tabs: "My Plugins" for managing installed plugins (enable/disable/remove) and "Plugins Workshop" for adding remote plugins by URL or browsing discoverable plugins.

### API

API key display with copy and regenerate buttons.
