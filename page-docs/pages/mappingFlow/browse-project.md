# Browse Project (`/project/$projectId`)

**Source**: `src/components/BrowsedProjectPage/`

The detail page for a project. Shows project information and its challenges.

## Sections

### Project Info

Displays the project name, icon, description, and metadata.

### Challenges List

A list of challenges belonging to the project using ChallengeCard components.

## Future

### Challenge Filters & Toggles

Add filters and toggles to help users narrow down the challenge list — e.g., by difficulty, completion status, work-on category, sort order, and a toggle to show/hide completed challenges.
